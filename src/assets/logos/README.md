# Logolar

Bu klasör (`src/assets/logos`), uygulama genelinde kullanılacak logolar, ikonlar ve marka görselleri için oluşturulmuştur.

## Kullanım
Logo dosyalarınızı (png, jpg, svg) buraya ekleyebilirsiniz.

Örnek import:
```javascript
import logo from '../assets/logos/logo.png'
```

## Önerilen Yapı
- **app-logo.png**: Uygulama ana logosu (Giriş ekranı, vb.)
- **sidebar-logo.svg**: Sidebar için vektörel logo
- **favicon.ico**: Tarayıcı sekmesi ikonu
